MediaOptimizer RDF Vocabulary (v1.0)

This namespace defines RDF terms used to describe media optimization metadata (e.g., optimizer tool, version, file sizes, optimization results).

Namespace URI

https://xandeyong.github.io/project/mediaoptimizer/ns/1.0/

Available Formats

  • Turtle (recommended)
  • RDF/XML

Example Usage

@prefix mo: <https://xandeyong.github.io/project/mediaoptimizer/ns/1.0/> .

<file1.jpg> mo:Optimizer_Tool "ffmpeg" ;
             mo:Optimize "true" ;
             mo:Original_Size "1543290" ;
             mo:Optimized_Size "1023480" .
